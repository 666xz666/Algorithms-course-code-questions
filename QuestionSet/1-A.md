####  Description

一只青蛙一次可以跳上1级台阶，也可以跳上2级。求该青蛙跳上一个n级的台阶总共有多少种跳法。

####  Input

多组测试样例。每组测试样例包含一个整数n。(1<=n<=100)

####  Output

每组测试样例输出一行，表示青蛙跳上n级台阶的跳法数量.  

所得到的结果模1000000007  

####  Sample Input
[Copy](javascript:CopyToClipboard\($\('#sampleinput'\).text\(\)\))

    
    
    3
    4
    

####  Sample Output
[Copy](javascript:CopyToClipboard\($\('#sampleoutput'\).text\(\)\))

    
    
    3
    5
